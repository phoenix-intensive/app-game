# Game

**Game** — это **одностраничное приложение (SPA)**, созданное с использованием **Angular**. Оно предоставляет интерактивный и удобный пользовательский опыт, 
включая функции проверки возраста, управление политикой использования cookies, игровой модуль и информационные страницы, 
такие как политика конфиденциальности и условия использования.

---

## Основные функции

- **Проверка возраста**: Обеспечивает доступ к приложению только пользователям, соответствующим требованиям по возрасту.
- **Управление cookies**: Пользователи могут принимать или отклонять cookies с помощью отображаемого баннера.
- **Игровой модуль**: Интерактивный раздел, где пользователи могут участвовать в основной игре.
- **Информационные страницы**:
  - Главная страница
  - Политика конфиденциальности
  - Условия использования
- **Адаптивный макет**: Универсальный заголовок и подвал для всех страниц.
- **Настройки для разных сред**: Разделение конфигураций для разработки и продакшена.

## Как запустить проект

### Требования

- **Node.js** (рекомендуемая версия: 18+)
- **Angular CLI** (рекомендуемая версия: 16+)

### Инструкция по запуску

1. Клонируйте репозиторий:
   ```bash
   git clone <ссылка на репозиторий>
   cd phoenix-intensive-app-game
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Запустите приложение:
   ```bash
   ng serve
   ```
   Приложение будет доступно по адресу `http://localhost:4200`.

4. Для сборки в режиме продакшена выполните:
   ```bash
   ng build --prod
   ```
   Готовые файлы будут находиться в папке `dist/`.

## Основные компоненты

- **Модуль приложения (`app.module.ts`)**: Корневой модуль, объединяющий компоненты, сервисы и маршруты.
- **Guard проверки возраста (`age.guard.ts`)**: Ограничивает доступ к определенным страницам для пользователей, не прошедших проверку возраста.
- **Общие компоненты макета**:
  - `HeaderComponent`: Содержит ссылки навигации и логотип.
  - `FooterComponent`: Включает дополнительные ссылки и информацию.
